# 🚖 Прогнозирование заказов такси (Анализ временных рядов)

Модель машинного обучения для прогнозирования количества заказов такси на следующий час в аэропортах с целью привлечения большего числа водителей в периоды пиковых нагрузок.

## 📋 Описание проекта
Исторические данные о заказах такси в аэропортах были проанализированы и подготовлены для построения модели прогнозирования временных рядов.    
Цель — добиться значения метрики **RMSE на тестовой выборке не больше 48**.

## 🛠 Технологии и стек
- **Язык:** Python
- **Обработка и анализ данных:** `pandas`, `numpy`, `statsmodels` (сезонная декомпозиция, тест Дики-Фуллера)
- **Машинное обучение:** `scikit-learn` (линейная регрессия, случайный лес, TimeSeriesSplit), `catboost`
- **Визуализация:** `matplotlib`, `seaborn`, `phik`

## 📊 Шаги реализации
1. **Ресемплирование:** Загрузка исторических данных и агрегация заказов с интервалом в один час.
2. **EDA и анализ:** Исследование трендов, сезонности и скользящих статистик.
3. **Генерация признаков (Feature Engineering):** Создание лаговых признаков (lag features) и скользящих средних для временных рядов.
4. **Обучение моделей:** Обучение нескольких моделей (линейная регрессия, случайный лес, CatBoost) с подбором гиперпараметров на тестовой выборке размером 10%.
5. **Оценка качества:** Проверка моделей на тестовых данных, достижение целевого порога RMSE (< 48) и оформление выводов.
